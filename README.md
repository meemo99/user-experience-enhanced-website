Dit [project](https://user-experience-enhanced-website-hm3f.onrender.com) is gemaakt voor de Buurtcampus. De uitgdaging is een digitale krant waarvan het doel gemeenschap bouwen centraal staat.
De Buurtcampus is werkzaam in 3 wijken in Amsterdam, de verhalen uit deze wijken kan je vinden op de website.
Elke wijk heeft zijn eigen pagina waarin alleen die artikelen te zien zijn.

## Installatie
<!-- Bij Installatie staat hoe een andere developer aan jouw repo kan werken -->
1. Ga naar nodejs.org en installeer Node.js 24.13.0 LTS (Long Term Support).
2. Fork de repository en open het project in VSCodium.
3. Open de terminal in VSCodium en voer het volgende commando uit:
   npm install
4. Start het project met:
   npm start
Het project is nu bereikbaar op http://localhost:8000. 

5. Wanneer je klaar bent, stop je de server met Control + C in de terminal.
